# udacity-portfolio
A collection of my data analytics and data science projects.

Welcome to my data analytics portfolio! This repository highlights my learning journey, coursework, and hands-on projects as I continue building my skills in data analysis, statistics, and programming.
I began my data analytics journey in 2022, when I completed the Google Data Analytics Certificate through Coursera. That experience helped me develop a strong interest in data analysis concepts, data cleaning, visualization, and working with real-world datasets. Wanting to deepen my technical and analytical skills, I went on to start my Bachelor’s degree in Data Analytics at Western Governors University (WGU) in 2023, with an expected graduation date of September 2026.
As part of my technical growth, I started learning SQL in 2023, focusing on querying and analyzing structured data, and began learning Python in 2024, where I’ve been applying programming concepts to data analysis, statistics, and exploration. I’m particularly interested in using data to answer meaningful questions and support informed decision-making.
So far, my projects include coursework from Udacity’s Introduction to Data Science, where I completed:

Investigate a Dataset – exploring, cleaning, and analyzing data to uncover trends and insights
Practical Statistics: Analyze A/B Test Results – applying statistical reasoning to evaluate experimental results

This portfolio will continue to grow as I complete additional coursework and personal projects. My goal is to build a strong, well-documented body of work that reflects both my technical development and my ability to think critically about data.
